Jekyll blog

新建站点

jekyll new mysite

安装缺少的gems

bundle install

命令启动本地预览服务器

bundle exec jekyll serve

推送blog到GitHub

进入你的 GitHub 仓库文件夹，添加生成的站点文件，并提交更改。然后，将更改推送到 GitHub：

git add .
git commit -m "Initial commit"
git push origin master
